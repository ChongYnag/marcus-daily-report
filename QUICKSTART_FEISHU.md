## 🔧 故障排查

### 没收到测试消息？

```bash
# 1. 检查配置文件
cat feishu_config.json

# 2. 手动测试
python3 -c "
import requests
import json
with open('feishu_config.json') as f:
    cfg = json.load(f)
r = requests.post(cfg['webhook_url'], 
    json={'msg_type':'text','content':{'text':'测试'}})
print(r.json())
"

# 3. 查看日志
cat reports/cron.log
```

### Webhook URL 无效？

- 确认是从飞书群机器人页面复制的
- 确认 URL 完整（没有截断）
- 确认机器人权限已开启

### 定时任务不执行？

```bash
# 检查 crontab
crontab -l

# 检查 cron 服务
sudo systemctl status cron    # Linux
sudo launchctl list | grep cron  # macOS
```
